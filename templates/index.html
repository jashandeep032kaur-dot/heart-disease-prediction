Heart Disease Prediction Project

This project predicts the risk of heart disease based on clinical and lifestyle data using Logistic Regression.

Dataset Columns

  • male: 1 if male, 0 if female
  • age: Age of the patient (years)
  • education: Years of education
  • currentSmoker: 1 if current smoker, else 0
  • cigsPerDay: Number of cigarettes per day
  • BPMeds: 1 if on blood pressure medication, else 0
  • prevalentStroke: 1 if had stroke before, else 0
  • prevalentHyp: 1 if has hypertension, else 0
  • diabetes: 1 if diabetic, else 0
  • totChol: Total cholesterol level
  • sysBP: Systolic blood pressure
  • diaBP: Diastolic blood pressure
  • BMI: Body Mass Index
  • heartRate: Resting heart rate
  • glucose: Blood glucose level
